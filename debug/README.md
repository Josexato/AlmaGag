# Debug

Esta carpeta contiene outputs de debug, screenshots y notas técnicas.

## Estructura

```
debug/
├── notes/         # Notas de debug y análisis de fixes
├── outputs/       # PNG y SVG de debug (gitignored)
└── screenshots/   # Screenshots de documentación
```

## Notas de debug

- `notes/FIX_SUMMARY.md` - Resumen de fixes aplicados
- `notes/HIERARCHICAL_LAYOUT_IMPROVEMENTS.md` - Mejoras de layout jerárquico

## Outputs

La carpeta `outputs/` contiene archivos PNG generados automáticamente por el sistema de debug.
Estos archivos están gitignored y se regeneran según sea necesario.

## Screenshots

Screenshots usados en documentación y reportes de issues.

**Nota:** Esta carpeta es principalmente para desarrollo. Los usuarios finales no necesitan interactuar con ella.
